karma-mocha-expect-sinon-example
================================

an example test setup for karma + mocha + expect + sinon.

Prerequisite
------------

Install node.js

Usage
-----

```
$ npm install
$ ./node_modules/karma/bin/karma start
```

Coverage output files are generated in ./coverage/ directory.
